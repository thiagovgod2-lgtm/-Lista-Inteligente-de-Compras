# 🛒 Lista Inteligente de Compras

Uma aplicação web simples e moderna para organizar suas compras de supermercado, farmácia e outras necessidades do dia a dia.

## ✨ Funcionalidades

- Adicionar produtos à lista
- Categorizar produtos
- Marcar itens como comprados
- Remover itens da lista
- Estatísticas em tempo real
- Sugestões automáticas de produtos frequentes
- Armazenamento local (LocalStorage)
- Interface responsiva
- Funciona offline
- Sem necessidade de servidor ou banco de dados

---

## 📷 Demonstração

### Tela Principal

- Adicionar produtos
- Selecionar categoria
- Visualizar lista de compras
- Acompanhar estatísticas

### Sugestões Inteligentes

O sistema analisa os produtos mais adicionados e exibe sugestões automáticas.
